Method

01

Observe, never run

The scanner reads bounded text files. It does not import, build, install, evaluate, or execute project code.

02

Arrange atomic evidence

A Cupboard piece is a donor observation connected to one host direction. Multi-project arrangements preserve every path, line, rule, and caution.

03

Steer explicitly

Breadth, novelty, compatibility, risk, focus, donor preferences, locks, and exclusions change deterministic ranking. They never invent evidence.

04

Keep claims bounded

“Fits” means the displayed snapshot signals align. It never claims buildability, API equivalence, license compatibility, or correctness.
