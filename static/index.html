Method

01

Look, never run

The scanner reads bounded text files. It does not import, build, install, evaluate, or execute project code.

02

Keep provenance

Every score component points to Evidence. Every Exhibit carries a Source Fingerprint over the files actually observed.

03

Suggest lightly

Affinities are directional: a Provision in one Exhibit may answer a Need in another. Recipes remain small and reviewable.

04

Stay local

The read-only server binds to loopback. Assets are bundled here; no remote requests, trackers, or write endpoints exist.
