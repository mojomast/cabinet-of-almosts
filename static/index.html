Method

01

Look, never run

The scanner reads bounded text files. It does not import, build, install, evaluate, or execute project code.

02

Explain every score

Scores are shown in plain English. They describe unfinished signals and reusable structure; they are not judgments of project quality.

03

Suggest lightly

The Workbench connects compatible evidence between historical projects. Every resurrection idea remains small, reviewable, and non-executing.

04

Stay local

The read-only server binds to loopback. Assets are bundled here; no remote requests, trackers, or write endpoints exist.
